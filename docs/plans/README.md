# Plans

Forge ticket lifecycle. Each ticket is a markdown file named `FRG-NNN-short-description.md`.

## Folders

| Folder | Status | Description |
|---|---|---|
| `active/` | In progress | Tickets currently being worked on by the Forge pipeline |
| `shipped/` | Merged | Tickets whose PR has been merged to main |
| `archived/` | Closed | Tickets that were cancelled, rejected, or abandoned |
